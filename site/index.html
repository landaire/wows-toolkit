WoWs Toolkit

Open-source desktop tool for World of Warships. Replay analysis, stat tracking, armor inspection, and game file browsing. Runs locally, doesn't modify game files.

Features

Replay Analysis

Full post-battle breakdown beyond what the game shows. Every player's damage dealt and received with type breakdowns (AP, HE, SAP, fire, flooding, torpedoes). View other players' builds and captain skills. See damage numbers the game hides from you. Export match data.

Stat Tracking

Performance tracking computed locally from your replay files. Damage, win rate, XP, per-ship breakdowns. No API keys or account linking needed.

Player Tracker

Logs every player you encounter in battle. See how often you face the same opponents, spot recurring division mates, build a local matchmaking history. All offline from replay files.

Armor Viewer

3D armor model inspection for any ship. Color-coded thickness visualization. Simulate artillery impacts at any angle and range against layered protection—deck plating, belt armor, citadel bulkheads, torpedo protection.

Resource Browser

Browse and extract game assets from packed IDX/PKG archives. Filter by path, view metadata, pull out textures, models, scripts, and config files. For modders, data miners, and the curious.
